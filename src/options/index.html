Bookmark RAG Manager

Embeddings for indexing; Q&A defaults to NVIDIA POST …/v1/responses with input + reasoning. Keys stay local.

Embeddings (index)

Provider root URL only — the extension calls POST {base}/{embeddings path} with { "model", "input": [...] } (same as OpenAI embeddings). Example base: https://integrate.api.nvidia.com/v1 or your NIM URL; if the host already ends with /v1, set the path below to embeddings only.

Embeddings path (optional)

Default: v1/embeddings. Use embeddings if your base URL already includes /v1.

Q&A (LLM)

Responses (NVIDIA) — POST …/v1/responses with { "model", "input": "…", "reasoning": { "effort": "…" } } (matches NVIDIA inference API). Optional: legacy Chat or Text completions.

Paths (optional)

Defaults: v1/responses, v1/chat/completions, v1/completions.

Prompt template (local RAG)

Default prompt already assumes bookmark search. Placeholders: {{context}} and {{question}} (or {{query}}).

Keyboard shortcut

Default: Ctrl+B (Windows/Linux) or Command+B (macOS).

To change it, open Chrome extension shortcuts.
